# Setting Up Email for Monarchy Homes Website

## What This Does

This guide helps you create a special "App Password" that allows the website to send automated emails (like download links) from hello@monarchyhomes.com - WITHOUT giving anyone your actual password.

**Your regular password stays private and unchanged.**

---

## Step-by-Step Instructions

### Step 1: Sign into Google

1. Open your web browser (Chrome, Edge, etc.)
2. Go to: **https://myaccount.google.com**
3. Sign in with **hello@monarchyhomes.com** if not already signed in

---

### Step 2: Turn on 2-Step Verification (if not already on)

**Check if it's already on:**
1. On the left side, click **Security**
2. Scroll down to "How you sign in to Google"
3. Look for **2-Step Verification**
4. If it says "On" - skip to Step 3
5. If it says "Off" - continue below:

**To turn it on:**
1. Click on **2-Step Verification**
2. Click **Get Started**
3. Enter your password when asked
4. Choose your phone number and click **Send**
5. Enter the code you receive via text
6. Click **Turn On**

✅ Done! 2-Step Verification is now enabled.

---

### Step 3: Create an App Password

1. Go to: **https://myaccount.google.com/apppasswords**
   
   (Or: Security → 2-Step Verification → scroll to bottom → App passwords)

2. You may need to sign in again

3. You'll see a page that says "App passwords"

4. Click the dropdown that says **Select app** and choose: **Mail**

5. Click the dropdown that says **Select device** and choose: **Other (custom name)**

6. Type: **Monarchy Homes Website**

7. Click **Generate**

8. **IMPORTANT:** You'll see a 16-character password like this:
   ```
   abcd efgh ijkl mnop
   ```

9. **Write this down or copy it somewhere safe** - you won't see it again!

10. Click **Done**

---

### Step 4: Send the Information Securely

Send the following to your web developer (via WhatsApp, text, or phone call - NOT email):

```
GMAIL EMAIL: hello@monarchyhomes.com
APP PASSWORD: [the 16 characters from Step 3, without spaces]
```

**Example:**
```
GMAIL EMAIL: hello@monarchyhomes.com
APP PASSWORD: abcdefghijklmnop
```

---

## What Happens Next

Your developer will add these details to the website settings. After that:

✅ The website can send download emails from hello@monarchyhomes.com  
✅ Customers see emails coming from your official address  
✅ Your actual password stays private  
✅ You can revoke this App Password anytime if needed  

---

## How to Revoke Access Later (if needed)

If you ever want to stop the website from sending emails:

1. Go to: **https://myaccount.google.com/apppasswords**
2. Find "Monarchy Homes Website" in the list
3. Click the **trash bin icon** next to it
4. Done - the app password no longer works
